# FirstLAVC.
## Краткое описаніе.
Программа написана на языкѣ С.  
Собравъ съ помощью Мейкфайла и запустивъ ее, можно провести одинъ изъ трехъ экспериментовъ, подавъ его число въ консоль. Эксперименты проверяютъ быстроту и точность воплощенныхъ въ файлахъ программы алгоритмовъ рѣшенія СЛАУ.
## Алгоритмы рѣшенія СЛАУ
### I. Обычный методъ Гаусса.
Функція метода ,,solveWithGaussianElimination" находится въ файлѣ ,,gaussian_elimination.c".  
Прямой ходъ метода начинаетъ съ верхней строки поданной на входъ расширенной матрицы СЛАУ.
