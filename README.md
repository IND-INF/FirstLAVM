# FirstLAVS.
## Краткое описаніе.
Программа написана на языкѣ С.  
Собравъ съ помощью Мейкфайла и запустивъ ее, можно провести одинъ изъ трехъ экспериментовъ, подавъ его число въ консоль. Эксперименты проверяютъ быстроту и точность воплощенныхъ въ файлахъ программы алгоритмовъ рѣшенія СЛАУ.
## Алгоритмы рѣшенія СЛАУ.
Алгоритмы программы рѣшаютъ лишь квадратныя СЛАУ, для матрицъ которыхъ существуетъ LU-разложеніе для сравнимости всѣхъ трехъ методовъ.
### 0. Доказательство необходимаго и достаточнаго условія существованія LU-разложенія.
Въ методахъ существованіе для матрицы LU-разложенія провіряется такимъ условіемъ: если и только если при шагѣ нумеръ n прямого хода метода Гаусса элементъ на пересѣченѣи строки нумеръ n и столбца нумеръ n равенъ нулю, у матрицы СЛАУ нѣту LU-разложенія.  
  
Докажемъ достаточность. Пусть разложеніе есть. а шагѣ нумеръ n, разсмотримъ главный миноръ получившейся матрицы размѣра n на n. Онъ равенъ нулю, такъ какъ имѣетъ нулевую нижнюю строку. Строки этого минора получились изъ строкъ соотвѣтствующаго минора исходной матрицы элементарными преобразованѣями, отсюда онъ тоже равенъ нулю. Соотвѣтствующая подматрица исходной матрицы вырождена, а также равна произведенію соотвѣтствующихъ подматрицъ L-матрицы и U-матрицы. Тогда одна изъ этихъ подматрицъ вырождена также, что, очевидно, невозможно, и поэтому разложенія не существуетъ.  

Докажемъ необходимость. Если ни при одномъ шагѣ элементъ не былъ равенъ нулю, то мы можемъ получить разложеніе описаннымъ ниже алгоритмомъ.
### 1. Обычный методъ Гаусса.
Функція метода ,,solveWithGaussianElimination" находится въ файлѣ ,,gaussian_elimination.c".  
Прямой ходъ метода начинаетъ съ самаго лѣваго столбца поданной расширенной матрицы СЛАУ.
